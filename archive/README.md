# archive/

ここにあるコネクタは **公式 MCP サーバーが提供されたため非推奨** です。

これらのサービスはベンダー公式（または公式に準ずる）MCP サーバーが利用できます。
そちらの方が認証・カバレッジ・保守の面で優れているため、**新規利用では公式 MCP を使ってください。**
対応する公式 MCP はリポジトリ直下の README「公式MCPが提供されているサービス」表を参照。

過去の参考実装・履歴保持のために残していますが、メンテナンス対象外です。

このリポジトリ本体（`/` 直下）は、**公式 MCP が存在しない日本 SMB 向け SaaS を中心とした独自実装**に純化しています。
